## 主要な導線（ユーザーフロー）

### 予約導線
```
ホーム → 宿泊予約 → [外部]各部屋予約ページ → 予約完了
```

### 情報収集導線
```
ホーム → スライドショー → 各特徴ページ → 詳細理解 → 予約検討
```

### アクセス導線
```
ホーム → アクセス → 交通手段確認 → 周辺スポット → 予約決定
```

### コンセプト理解導線
```
ホーム → エコホテル/空間デザイン/過ごし方 → 価値理解 → 予約
```
