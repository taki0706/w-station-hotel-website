# 案D: 指定トップ構成ベース再提案

作成日: 2026-02-13  
基準構成: `Hero → Story&Concept → Rooms → Feature → Footer`

## 1. トップページ構成（指定構成に準拠）

### Header（全ページ共通）
- 目的: どのページからでも `予約` と `主要導線` に最短到達させる。
- 要素:
  - 固定CTAボタン: `空室検索・予約`
  - 言語切替: 選択式ボタン（`JP / EN`）
  - 展開メニュー（Desktop: ドロップダウン / Mobile: ハンバーガー）
- 展開メニュー項目（推奨）:
  - `Story&Concept`
  - `Rooms / 予約`
  - `Feature`
  - `Access`
  - `お問い合わせ`
- 振る舞い:
  - スクロール時はヘッダー固定（sticky）でCTAを常時表示。
  - 言語切替は現在ページの対応言語URLへ遷移。
  - メニュー最下部にも `予約CTA` を再掲して離脱を防ぐ。


### Hero
- 背景: 代表ビジュアル1枚 + 補助スライド2〜3枚
- 訴求文: 「駅前・アート・サーフ/ワーク対応・ZEB」を1文で要約
- CTA:
  - `空室を確認する` → `予約ページ`

### Story&Concept（トップ内セクション）
- 掲載要素:
  - `P014` 世界のKEN（ホテルの背景）
  - `P008` 過ごし方（朝/昼/夜）
  - `P006` エコホテル
  - `P007` 空間デザイン
- ボタン: `Story & Conceptを詳しく見る` → `Story&Conceptページ`
- 追記要素: `おすすめ宿泊プラン`（後述）

### Rooms（トップ内セクション）
- 8室をカード表示（画像、部屋名、定員、ベッド構成、価格目安）
- カード押下: `予約ページ` へ遷移
- サブCTA:
  - `客室比較を見る` → 同ページ内比較表
  - `予約前FAQを見る` → 同ページ内FAQ

### Feature（トップ内セクション）
以下の内容を統合・整理して表示
  - `P009` 照明とデザイン
  - `P010` 銭湯絵師・中島盛夫画
  - `P011` 画廊&アート作品
  - `P012` オープンダイニングキッチン
  - `P013` New York (入浴) BAR
- ボタン: `Featureを詳しく見る` → `Featureページ`

### Footer
- 既存フッター要素を維持しつつ、以下を追加
  - `Story&Concept`
  - `Feature`
  - `Access`
  - `お問い合わせ`
  - `運営会社`
- 固定CTA（予約/電話）は維持
